# BI Financeiro - Empresa de Distribuição de Alimentos
*Meu primeiro projeto de Business Intelligence aqui no GitHub. 
Todos os dados apresentados são fictícios de uma empresa fictícia, apenas para fins de demonstração e apresentação do meu porttfolio.*

## Análise Feitas Nesse Dashboard

### Os objetivos da análise de BI nesse projeto são para obter os seguintes conhecimentos e resultados do ano de 2019:

- #### Nessa primeira página, são apresentados valores de Receita, Custos e Margem, nela podemos ver as seguintes informações:
1. Receita, margem  bruta e % de margem bruta por mês;
2. % da fatia da receita por linha de produto;
3. Margem bruta por fornecedor;
4. Análise por equipe de vendas e por canal de venda.
